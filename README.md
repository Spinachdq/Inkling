# 碎片化内容整理网站

> **笔落皆所思，寸墨藏山河**

一款轻量型笔记工具的前端原型，用于快速导入、解析外部内容，通过 AI 辅助与手动编辑将零散收藏转化为结构化笔记，以卡片形式呈现。

---

## 如何运行

本项目为纯前端静态页面，无需构建，用浏览器打开即可：

1. **方式一：直接打开**  
   双击 `index.html` 用默认浏览器打开。

2. **方式二：本地服务器（推荐，避免部分浏览器的跨文件限制）**  
   在项目根目录执行：
   ```bash
   # 若已安装 Node.js
   npx serve .

   # 或使用 Python
   python -m http.server 8080
   ```
   浏览器访问：`http://localhost:3000` 或 `http://localhost:8080`。

## 功能概览

| 功能 | 说明 |
|------|------|
| **随手记** | 创建纯原创笔记，支持标题、正文、加粗/斜体、AI 生成大纲与提炼核心（模拟） |
| **链导入** | 弹窗输入链接，模拟解析公众号/小红书等，跳转编辑页加工 |
| **文上传** | 拖拽或选择 Word / PDF / TXT，模拟提取正文后进入编辑 |
| **AI 创作** | 对话式 AI 页面，模拟回复；支持复制到剪贴板、复制到笔记 |
| **卡片区** | 按日期/分类排序，点击进详情，hover 显示快捷编辑、删除 |
| **编辑页** | 标题、正文、工具栏（加粗、斜体、AI 大纲、AI 核心）、删除，约 30 秒自动存草稿 |

数据保存在浏览器 `localStorage`，关闭或清空数据后会重置；链接解析、文档解析、AI 能力为**前端模拟**，真实产品需接入后端与 AI 服务。

## 设计说明

- **色彩**：低饱和暖色（米白、浅灰、淡蓝、淡粉），标签与功能键用柔和辅助色。
- **字体**：Noto Sans SC / 微软雅黑，slogan 约 18–20px，正文 14px，标签/日期 12px。
- **布局**：上（slogan）— 中（4 个圆形功能键）— 下（卡片网格），响应式 1–5 列。

符合 PRD 2.0 中的产品定位与视觉规范，便于后续接入真实解析与 AI 接口。
